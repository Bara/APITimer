Timer-Mapzones

This plugin simply manages mapzones.

Commands:
sm_restart - Restart timer
sm_r - Restart timer
sm_start - Teleport back to start/spawn
sm_s - Teleport back to start/spawn
sm_bonusstart - Restart bonus timer
sm_br - Restart bonus timer
sm_b - Restart bonus timer
sm_levels - Stage teleport menu
sm_stage - Stage teleport menu
sm_stuck - Teleport to last stage (penalty time)

Admin Commands:
sm_zoneadmin - Displays zone admin menu
sm_zoneadminmode <0|1> - Enable/Disable zone admin mode (admins will not trigger zones)
sm_zoneid <id> - Set zone ID (Stage ID)
sm_zonetype <type> - Overwrite zone type
sm_zonereload - Reload zones
sm_npc_next - Spawn NPC teleporter
sm_zone - List zones and let admins teleport to them

Config file in cfg/sourcemod/timer/timer-mapzones.cfg
