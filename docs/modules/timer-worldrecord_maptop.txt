Timer-Worldrecord_Playerinfo

This module allows to display top records for a given map.

Commands:
sm_mtop - Displayer top records for current map
sm_mtop <map> - Displayer top records for a given map
sm_mbtop - Displayer top bonus records for current map
sm_mbtop <map> - Displayer top bonus records for a given map
sm_mstop - Displayer top short records for current map
sm_mstop <map> - Displayer top short records for a given map
