Timer-JStats

This plugin provides long jump stats.

Commands:
sm_lj - Toggle long jump mode
sm_ljblock - Register destination
sm_ljsound - Toggle sounds
sm_ljpopup - Toggle popup
sm_gap - Measure units between 2 points
sm_ljtop - Displays top ling jumps

Admin Commands:
sm_ljadm - Delete long jumps


Config file in cfg/sourcemode/timer/timer-ljstats.cfg
